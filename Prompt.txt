✅ Programme photobooth créé
J’ai généré une application complète en Python pour Windows avec écran tactile :

main.py : orchestration principale
camera_agent.py : gestion de la caméra / capture
photo_agent.py : sauvegarde et overlay photo
print_agent.py : impression Windows
ui_agent.py : interface tactile PySide6
requirements.txt : dépendances
README.md : instructions d’installation et d’usage

💡 Notes
Le programme utilise la caméra par défaut (camera_index=0).
Les photos sont sauvées dans le dossier photos/.
L’impression Windows nécessite pywin32 et une imprimante par défaut configurée.
